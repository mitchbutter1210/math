Overview

A program built to do basic math and advanced formulas including:

  • Ohm's Law
  • Basic Arithmetic
  • Powers
  • Square Roots
  • Geometry Distance Formula
  • Quadratic
  • Density Formula

To install:

  1. Click download zip if you are on Windows
  2. Extract the file
  3. Open the newly created folder and run Math.jar
